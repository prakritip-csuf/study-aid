## StudyAid - Smart Learning Platform
StudyAid is a full-stack learning web application designed to help students learn effectively using digital flashcards, quizzes, and progress tracking.  It provides a clean and interactive interface where users can create their own study materials and monitor their learning journey.

## Project Overview: 

### The Stuy-aid Web Application allows users to:
- Create flashcard sets
- Add Title and descriptions
- Open a set and create flashcards inside it
- Update & delete cards
- Support up to 50 cards per set
- Track the progress
- Take quiz based on the flashcards users make
- Records the best score for the user for each set

### User Authentication
- Login/Logout system  
- Users only see **their own flashcard sets and progress**  
- Backend-secured routes with token/session handling

### The system is built using:
- Frontend: React + JavaScript
- Backend: Python Flask API
- Database: SQLite via SQLAlchemy ORM
